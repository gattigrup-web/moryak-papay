
# SeaJobs — срочные вакансии для моряков

Просмотр вакансий — без регистрации. Публикация предложений — только после регистрации по email.

## Запуск локально
1. Скачайте файлы (`index.html`, `styles.css`, `app.js`, `README.md`) в одну папку.
2. Откройте `index.html` в браузере.

## Публикация на GitHub Pages
1. Создайте репозиторий на GitHub.
2. Загрузите файлы в корень репозитория.
3. В настройках репозитория включите **Pages**: Source — `main` (или ваша ветка), папка — `/root`.
4. Через минуту сайт будет доступен по адресу GitHub Pages вашего репозитория.

## Заметки по безопасности и продакшену
- В демо авторизация хранится в `localStorage` и не подходит для продакшена.
- Для реальной публикации предложений используйте бэкенд (например, Firebase/Auth + Firestore, Supabase, либо простой Node/Express API).
- Валидация и антиспам: лимит частоты публикаций, проверка email, модерация.
- Не храните пароли в открытом виде — используйте хеширование на сервере.
